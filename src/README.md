## 🛠 安装

首先clone仓库：

````bash
git clone https://github.com/wuliang142857/my-cookbook.git
````

安装依赖：

<CodeGroup>
<CodeGroupItem title="yarn">
```bash
yarn install
```
</CodeGroupItem>

<CodeGroupItem title="npm">
```bash
npm install
```
</CodeGroupItem>
</CodeGroup>
